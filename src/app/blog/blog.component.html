Shattering.. piggies

1/20/2022

making objects shatter in unity is very simple and satisfying. simply take your base model and create separations along edges in blender.

when the pig enemy is shot, the base entity model is destroyed and in its place I instantiate the separated model. the separated model is a rigid body. add some force to the rigid body on instantiation for better effects.

Definitely could add some more variation in the "shattering" here. I guess this really isn't shattering to begin with, I just thought it was pretty funny/eerie how their faces just popped off. Also wondering if I could create some script in blender to develop these shatter patterns for me randomly. Some sort of algorithm to traverse the edges... The hardest part I imagine would be making high resolution shatters and figuring out how to create many closed polygon separations/edge loops that don't intersect.

Point and Ripple

12/5/2022

Trying to create a ground warping effect in my game. To be used when a boss stomps or hops around the level. Thinking about implementing This with a vertex shader on the floor material... some sort of point and ripple effect. Probably the most common example of this is with water ripple shaders, but I'm trying to create something tighter and more abrasive.

I have this ripple effect running in a vertex shader. The big G Boss casts a ray into the ground, on hit the coords put into the vertex shader via uniforms. The "ripple effect" function I'm using is based off of this sombrero?? function. I run the vertices through this based on their distance to the origin of impact. Over time, the wave is dampened until there's basically no movement. I haven't been able to spend too much time optimizing this, I think I just got this basic ripple after a few days of trying to understand the different coordinate systems... so that my impact input (world) coordinate aligned with the vertices. But ideally the wave would work inverse, the points farthest from the impact will be the most unstable as if to create a crater. I also want to continue devloping this function to see if I can get it to work on more complicated surfaces instead of just flat planes.

Bubbles Screensaver

10/5/2022

Experiments in creating a function for a bubble screensaver. I think I spent the whole month trying to figure out the best process for illustrating bubbles before I even touched a bubble screensaver.

Ultimately don't feel like I have much to show with these illustrations, I was just so fried this month this is all I could focus on. I think I was primarily inspired by goofy anthropomorphized bubble machine photos.

I then whipped these bubbles up in after effects. Just played around trying to make an expression that would make it easy to build these bubble screensavers.

Nothing too crazy going on here, but fun to consider all of the variables to make this as modular as possible. I've set the initial positions to custom sliders (variables) and have their positions follow a triangle wave over time. this way when they reach the ~amplitude~/bounds of the composition their directions flip as if they were bouncing off of a surface.. Currently I'm not handling the composition size, I'm also hardcoding in speeds when that could also be attached to a slider.

ultimately this was a stupid project I spent a lot of time on in a hyperfixated, fried kind of way. looks kind of cute and makes for easy overlays when I'm being lazy trying to build other compositions.
